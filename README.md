### 🚀 Proyectos Destacados

> **Nota para reclutadores:** Estos son los 3 proyectos donde he aplicado patrones de diseño avanzados, optimización de estado y sincronización de datos resiliente.

#### 1. Gesti-n-Usuarios 📱
* **Descripción:** Aplicación de gestión de usuarios. El proyecto consume datos de la API randomuser.me y los presenta en una tabla interactiva, permitiendo ordenar, filtrar, eliminar y restaurar usuarios, todo con un sistema de historial de acciones y notificaciones en tiempo real.
* **Reto técnico resuelto:** Implementación de patrones de diseño para crear una aplicación que permita funcionalidades desacopladas con objetos con responsabilidades únicas, con un código muy facil de entender y de extender.
* **Tech Stack:** React19, Vite, PNPM, TypeScript, Custom Hooks, ESlint, Tailwind CSS
* [🔗 Código en GitHub]([link-al-repo](https://github.com/Jean7Pierre/Gesti-n-Usuarios)) | [🌐 Demo en Vivo]([link-a-la-demo](https://gest-n-users.netlify.app/))

#### 2. To-do App ⚽
* **Descripción:** Aplicación de lista de tareas (To-do MVC) construida con tecnologías de vanguardia.
* **Reto técnico resuelto:** Implementacion de caracteristicas como Deshacer las acciones realizadas por el usuario, implementacion de reducers, uso de custom hooks para abstracción de la logica de los componentes.
* **Tech Stack:** JavaScript (ES6+), React, Vite, Framer Motion, ESlint, Tailwind CSS
* [🔗 Código en GitHub]([link-al-repo](https://github.com/Jean7Pierre/to-do-app)) | [🌐 Demo en Vivo]([link-a-la-demo](https://hilarious-trifle-bd5b18.netlify.app/))
